Information and Communication Technology (ICT)
Internship Program

This website hosts the ICT educational program for interns from the Monrovia Consolidated School System (MCSS) working with NOVA Web Development.

We are beginning with an introduction to web design and development, stressing a working knowledge of HTML and CSS. Students will learn to create syntactically valid html 5 web pages using html 5 elements, and to style them with syntactically valid CSS 3. Advanced students will be encouraged to purse further study in JavaScript and Python.
