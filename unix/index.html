This tutorial on the Unix command-line interface is designed to provide the aspiring web developer with just enough skills to manage files and directories through a Unix shell for the purpose of creating and maintaining a website.

To enable practice with remote server features such as ssh, scp, sftp, rsync, and setting up a web server, all lessons in this tutorial have been tested on a Raspberry Pi computer running Raspberry Pi OS on a home network set up as a web, ssh, and rsync server.

After completing this tutorial, you will be ready to start learning HTML using Getting Down with HTML.
