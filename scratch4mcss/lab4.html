Scratch for MCSS

Lab 4: Looping and Drawing Your Own Sprites

Welcome to lab four of our Scratch class! Today we will learn how to write a loop that will make a sprite walk across the screen.

We will also learn how to use Scratch's built-in graphics editing tools to draw our own sprites.

Vocabulary

raster graphics
Images that are represented as a matrix (rows and columns) of pixels (picture elements, or little colored lights on the screen).
vector graphics
Images that are represented by mathematical descriptions of geometric shapes (rectangles, circles, polygons).

After talking about these important ideas, we will work on a new project that will make a sprite walk across the screen using a loop.

Activity:

  1. Download Walking Jeff.
  2. Run the program.
  3. Study the code to learn how it works.
  4. Make a program of your own that:
    • Is named "[Your name]Walking".
    • Has your head on a walking sprite body.
    • Uses a loop to make the sprite walk.
    • Says something about you.
