Lab 6: Variables

Welcome to lab six of our Scratch class! Today we will learn how to use variables to store values.

We have now used all three of the programming control structures, sequencing, selection, and looping. In this program we focus on data. How can we save information in our program to be use it later? The answer is variables, which are names that reference values our programs can store and retrieve.

After reviewing our vocabulary, we will see a new program: Estimation on a Number Line.

Vocabulary

raster graphics
A method of representing images as 2 dimensional arrays of colored dots or pixels (picture elements).
vector graphics
A method of representing images as mathematical representations of geometric objects.
variable
A name which refers to a stored value in a computer.

Activity:

  1. Download Estimation on a Number Line.
  2. Run the program.
  3. Study the code to learn how it works.
  4. Identify the variables in the program.
  5. Identify the control structures used in the program.
  6. Create your own program that uses a variable.
