Scratch for MCSS

Lab 5: Tomato Toss

Welcome to lab five of our Scratch class! Today we will learn how to use selection to keep score in a game.

We have now used all three of the programming control structures, sequencing, selection, and looping. This program uses all three control structures. See if you can find where in the code.

After reviewing our vocabulary, we will work on a new project, making our first game.

Vocabulary

bit
A binary digit, which is usually represented as a 0 or a 1. When it comes to computers, it's all just bits!
variable
A name which refers to a stored value in a computer.
IT
IT stands for information technology.
ICT
ICT stands for information and communications technology.

Activity:

  1. Download Tomato Toss.
  2. Run the program.
  3. Study the code to learn how it works.
  4. Identify examples of sequencing, selection, and looping in the program.
  5. Make a program of your own that uses all three control structures.

This program also uses a programming idea we haven't talked about yet - a variable. Look at the definition of variable above and see if you can find where a variable is used in this program.
