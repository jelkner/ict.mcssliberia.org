Scratch for MCSS

Day 2: Message Passing

Welcome to lab two of our Scratch class! Today we will learn about three very important concepts in Scratch programming:

  1. event-driven programming
  2. message passing
  3. sequencing

After talking about these important ideas, we will work on our 2nd project, Name Dance. In addition to message passing between sprites to control action, we can use this project to explore several of the effects available through blocks in the Looks palette and introduce graphics features in Scratch.

Activities:

  1. Create your second Scratch project: Name Dance.
