Lab 7: Programming Challenges

Welcome to lab seven of our Scratch class! Today you will be given a few programming challenges to see how well you can apply what you have learned.

We have seen all three of the programming control structures, sequencing, selection, and looping, and we have been introduced to variables that enable us to give names to values in our programs so we can store them and retrieve them later.

Using the following programs:

  • Binary Converter
  • Binary Challenge
  • Binary 255
  • Jelkner's Dec to Bin

do each of the following:

  1. Study each of the programs. What do they do? How do they do it? Can you identify the section of code used to compute the binary number output from the decimal number input? Do all of the programs use the same algorithm to do this?
  2. Create a Scratch program that let's the user convert from Liberian dollars (LRD) to US dollars (USD).
  3. Change BinaryConverter.sb3 to use the shorter algorithm to convert decimal to binary that JelknerDecToBin.sb3 uses.
