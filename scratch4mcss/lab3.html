Lab 3: Programming Control Structures

Welcome to lab 3 of our Scratch class! Today we will learn about the 3 control structures in computer programs.

Vocabulary

sequencing
Arranging instructions in order to accomplish a task step-by-step.
selection
Executing instructions based on the answer to a question.
looping
Executing instructions again and again as long as something is true.

We will start class today with a short video that introduces these three control structures.

After talking about these important programming concepts, we will study a program written in by someone else, and identify each of the control structures in that program.

Activity:

  • You will work in groups of three for this project.
  • Copy the Scratch project: Virtual Pet - Reptile to your networked computer.
  • Load it in Scratch and run it, taking notes on what it does.
  • Try to find examples of sequencing, selection, and looping in the program.
  • Prepare a presentation to the class on what you have learned.
