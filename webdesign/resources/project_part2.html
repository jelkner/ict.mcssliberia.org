Web Page Design Project: Part 2

Overview

Now that you have completed Lesson 3: Basic Selectors from Getting Down with CSS, it is a good time to apply your new CSS skills to improving the look of the project you made at the end of Getting Down with HTML.

Concept Review

You should now know the following CSS selectors, and be able to use them:

  • element selectors
  • class selectors
  • id selectors
  • group selectors

What Do I Need to Do?

  1. Use CSS to style your project. To earn an A you should use each of the following:
    1. element selector
    2. group selector
    3. class selector
    4. id selector
    5. colors for backgrounds, heading, links, etc.
    6. sizes for headings, paragraph text, link items, etc.
  2. Add some beautiful Google web fonts to your document using these instructions. You can look at the source to my home page to see another example using three Google web fonts.
